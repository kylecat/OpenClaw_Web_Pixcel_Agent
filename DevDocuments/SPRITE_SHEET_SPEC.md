# Sprite Sheet Specification — OpenClaw PixelAgent

> 版本：1.0 | 日期：2026-03-05

本文件定義角色與等距場景素材的精確規格，供像素畫師製作最終版素材。

---

## 1. 室內角色 Sprite（Top-Down，現行格式）

### 1.1 基本參數

| 項目 | 值 |
|------|-----|
| 來源幀尺寸 | **16 × 32** px |
| 顯示縮放 | 4× → **64 × 128** px |
| 每行幀數 | 7（cols 0–6） |
| 行數 | 3 |
| Sprite Sheet 尺寸 | **112 × 96** px |
| 格式 | PNG, RGBA（透明背景） |
| 色彩深度 | 32-bit RGBA |

### 1.2 Sheet 佈局

```
         Col 0    Col 1    Col 2    Col 3    Col 4    Col 5    Col 6
        ┌────────┬────────┬────────┬────────┬────────┬────────┬────────┐
Row 0   │ Walk 0 │ Walk 1 │ Walk 2 │ Walk 3 │  (opt) │  (opt) │  (opt) │  ← DOWN (面向螢幕)
(DOWN)  │  16×32 │ =IDLE  │  16×32 │  16×32 │        │        │        │
        ├────────┼────────┼────────┼────────┼────────┼────────┼────────┤
Row 1   │ Walk 0 │ Walk 1 │ Walk 2 │ Walk 3 │  (opt) │  (opt) │  (opt) │  ← UP (背對螢幕)
(UP)    │        │ =IDLE  │        │        │        │        │        │
        ├────────┼────────┼────────┼────────┼────────┼────────┼────────┤
Row 2   │ Walk 0 │ Walk 1 │ Walk 2 │ Walk 3 │  (opt) │  (opt) │  (opt) │  ← RIGHT（LEFT = 水平翻轉）
(RIGHT) │        │ =IDLE  │        │        │        │        │        │
        └────────┴────────┴────────┴────────┴────────┴────────┴────────┘
```

- **Cols 0–3**：行走動畫幀（循環播放 0→1→2→3→0…）
- **Col 1**：同時作為 **IDLE**（靜止）幀
- **Cols 4–6**：保留，可用於特殊動畫（目前留空 / 透明）
- **LEFT 方向**：程式碼水平翻轉 Row 2（RIGHT），**不需額外繪製**

### 1.3 行走動畫週期

```
Frame 0: 左腳前伸      ←  起始
Frame 1: 雙腳並齊      ←  IDLE 靜止幀
Frame 2: 右腳前伸
Frame 3: 雙腳並齊（回程）
```

幀率：**~8.3 fps**（`FRAME_DURATION = 0.12s`）

### 1.4 角色設計要點

| 部位 | Gaia | Astraea |
|------|------|---------|
| 整體配色 | 綠色系 | 紫色系 |
| 髮色 | 深綠 `#3C8C5A` | 深紫 `#5A46A0` |
| 上衣 | 亮綠 `#50B478` | 亮紫 `#7864C8` |
| 褲子 | 暗綠 `#3C5A3C` | 暗紫 `#463278` |
| 膚色 | 桃色 `#FFDCB9` | 桃色 `#FFDCB9` |
| 鞋子 | 棕色 `#503C28` | 棕色 `#503C28` |
| 身高比例 | 頭:身 ≈ 1:3（標準 RPG） | 同左 |

## 2. 等距角色 Sprite（Isometric，8 方向）

### 2.1 基本參數

| 項目 | 值 |
|------|-----|
| 來源幀尺寸 | **16 × 24** px |
| 顯示縮放 | 4× → **64 × 96** px |
| 每行幀數 | 4（walk frames） |
| 行數 | 8（directions） |
| Sprite Sheet 尺寸 | **64 × 192** px |
| 格式 | PNG, RGBA（透明背景） |
| 色彩深度 | 32-bit RGBA |

### 2.2 Sheet 佈局

```
         Col 0    Col 1    Col 2    Col 3
        ┌────────┬────────┬────────┬────────┐
Row 0   │ Walk 0 │ Walk 1 │ Walk 2 │ Walk 3 │  ← S  (正南，面向螢幕下方)
(S)     │  16×24 │ =IDLE  │  16×24 │  16×24 │
        ├────────┼────────┼────────┼────────┤
Row 1   │ Walk 0 │ Walk 1 │ Walk 2 │ Walk 3 │  ← SW (西南)
(SW)    │        │        │        │        │
        ├────────┼────────┼────────┼────────┤
Row 2   │ Walk 0 │ Walk 1 │ Walk 2 │ Walk 3 │  ← W  (正西)
(W)     │        │        │        │        │
        ├────────┼────────┼────────┼────────┤
Row 3   │ Walk 0 │ Walk 1 │ Walk 2 │ Walk 3 │  ← NW (西北)
(NW)    │        │        │        │        │
        ├────────┼────────┼────────┼────────┤
Row 4   │ Walk 0 │ Walk 1 │ Walk 2 │ Walk 3 │  ← N  (正北，背對螢幕)
(N)     │        │        │        │        │
        ├────────┼────────┼────────┼────────┤
Row 5   │ Walk 0 │ Walk 1 │ Walk 2 │ Walk 3 │  ← NE (東北)
(NE)    │        │        │        │        │
        ├────────┼────────┼────────┼────────┤
Row 6   │ Walk 0 │ Walk 1 │ Walk 2 │ Walk 3 │  ← E  (正東)
(E)     │        │        │        │        │
        ├────────┼────────┼────────┼────────┤
Row 7   │ Walk 0 │ Walk 1 │ Walk 2 │ Walk 3 │  ← SE (東南)
(SE)    │        │        │        │        │
        └────────┴────────┴────────┴────────┘
```

### 2.3 方向定義（等距投影）

```
         N (row 4)
        ╱ ╲
  NW (3)    NE (5)
  │              │
  W (2)      E (6)
  │              │
  SW (1)    SE (7)
        ╲ ╱
       S (row 0)
```

- **S**：角色面向螢幕下方（最常見，正面全身）
- **N**：角色背對螢幕（背影）
- **E / W**：側面，身體微斜向等距透視
- **NE / NW / SE / SW**：45° 斜角，身體帶等距傾斜

### 2.4 等距角色設計要點

| 項目 | 規格 |
|------|------|
| 身高比例 | 頭:身 ≈ **1:2**（chibi / Q版，比室內略矮） |
| 頭部 | 佔幀高度上 1/3（~8px），圓潤 |
| 身體 | 佔中間 1/3（~8px），略寬扁 |
| 腿+腳 | 佔下 1/3（~8px） |
| 身體傾斜 | NE/E/SE 方向身體中心右偏 1px；NW/W/SW 左偏 1px |
| 腳部基準 | 角色「腳底」對齊幀底部 row 23，用於 depth sort 錨點 |
| 配色 | 與室內版**完全一致**（見 §1.4 表格） |
| 輪廓線 | 1px 深色輪廓（#2A2A2A），增加等距場景辨識度 |

### 2.5 行走動畫週期

與室內版相同的 4 幀循環，但手腳擺動幅度可略小（chibi 比例）：

```
Frame 0: 左腳前伸 + 右臂前擺
Frame 1: 雙腳並齊（IDLE）
Frame 2: 右腳前伸 + 左臂前擺
Frame 3: 雙腳並齊（回程）
```

### 2.6 Phase 1 過渡（4方向映射）

在 8 方向素材完成前，渲染器使用既有 4 方向室內 sprite + 方向重映射：

| Grid 方向 | 等距螢幕方向 | 使用的室內 Row |
|-----------|-------------|---------------|
| UP        | NW          | Row 1 (UP)    |
| DOWN      | SE          | Row 0 (DOWN)  |
| LEFT      | SW          | Row 2 (flip)  |
| RIGHT     | NE          | Row 2 (RIGHT) |

## 3. 等距地面 Tile

### 3.1 基本參數

| 項目 | 值 |
|------|-----|
| Tile 尺寸 | **64 × 32** px（2:1 鑽石） |
| Grid 尺寸 | 16 × 12 tiles |
| 投影 | 標準等距（x 軸 2:1 傾斜） |
| 格式 | PNG, RGBA |

### 3.2 Tile 種類

| 檔名 | 描述 | 可行走 | 參考色 |
|-------|------|--------|--------|
| `iso_grass.png` | 草地 | Yes | `#64B450` |
| `iso_dirt.png` | 泥地 | Yes | `#A0825A` |
| `iso_path.png` | 小徑 | Yes | `#BEAA8C` |
| `iso_water.png` | 水 / 河流 | **No** | `#4682C8` |

### 3.3 鑽石形狀定義

```
          (32, 0)
           ╱╲
          ╱  ╲
         ╱    ╲
(0, 16) ╱      ╲ (64, 16)
         ╲      ╱
          ╲    ╱
           ╲  ╱
            ╲╱
          (32, 32)
```

- 所有 tile 為鑽石外框內填色
- 可加入細微紋理（草地：隨機深淺綠點；泥地：小石子點）
- 邊緣保持 1px 透明，讓相鄰 tile 有自然分隔

## 4. 等距建築

### 4.1 總覽

| 檔名 | 描述 | 佔地 | 圖片尺寸 | 互動 |
|-------|------|------|----------|------|
| `greenhouse.png` | 溫室（玻璃屋） | 2×2 tiles | ~128 × 112 px | 點擊開啟面板 |
| `weather_station.png` | 氣象站 | 1×1 tile | ~64 × 88 px | 點擊開啟面板 |
| `cabin.png` | 小木屋（資材室） | 2×2 tiles | ~128 × 116 px | 點擊切場景 |

### 4.2 設計準則

- **透視一致**：左上光源，右下陰影
- **高度表現**：建築高度以「牆面像素」表達，屋頂為鑽石頂面
- **風格**：像素風格，低飽和度但可辨識
- **互動提示**：可互動建築在 hover 時加白色鑽石外框（由程式處理，不需畫在素材內）

### 4.3 溫室設計提示

```
        ╱────────╲
       ╱ 玻璃屋頂 ╲        ← 半透明綠玻璃
      ╱────────────╲
     │ ┌──┐  ┌──┐ │        ← 玻璃面板 + 金屬框架
     │ │🌱│  │🌿│ │        ← 室內植物隱約可見
     │ └──┘  └──┘ │
     └──────────────┘
```

### 4.4 小木屋設計提示

```
        ╱╲
       ╱屋╲                ← 三角木屋頂（深棕）
      ╱  頂╲
     ╱──────╲
     │ ┌──┐ │              ← 木牆 + 門
     │ │門│ │
     └─┴──┴─┘
```

## 5. 植被與裝飾

| 檔名 | 描述 | 尺寸 | 可行走 |
|-------|------|------|--------|
| `tree_1.png` | 闊葉樹 | 32 × 48 px | No |
| `tree_2.png` | 針葉樹 | 32 × 52 px | No |
| `bush.png` | 灌木叢 | 24 × 20 px | No |
| `crop_plot_empty.png` | 空農田（鑽石） | 64 × 32 px | Yes |
| `crop_plot_growing.png` | 生長中農田 | 64 × 32 px | Yes |
| `crop_plot_ready.png` | 可收成農田 | 64 × 32 px | Yes |

### 5.1 樹木設計

- 基座對齊等距 tile 底部中心
- 樹冠向上延伸（不受 tile 約束）
- 樹冠用深度排序的「腳部 row」決定繪製順序
- 深淺兩種綠色區分不同樹種

### 5.2 農田設計

- 基於鑽石 tile 形狀
- 泥地底色 + 階段對應植物：
  - **Empty**：翻土紋理
  - **Growing**：小綠苗
  - **Ready**：成熟作物（帶果實色彩點綴）

## 7. 技術約束摘要

| 約束 | 說明 |
|------|------|
| 像素對齊 | 所有像素必須對齊整數座標，不可有 sub-pixel |
| 透明背景 | 所有素材 PNG 使用 RGBA，背景透明 |
| 無抗鋸齒 | 像素風格不使用 anti-aliasing / 半透明邊緣 |
| 光源方向 | 統一**左上方** |
| 調色板 | 自由，但同一角色室內/室外配色需一致 |
| Canvas rendering | 像素素材：`imageSmoothingEnabled = false`；高解析度素材：`true` |
| 深度排序錨點 | 角色/建築以「腳部所在 row」為 depth sort 的 key |

## 8. Placeholder 生成工具

開發期間可使用 placeholder 素材先行開發，最終替換為畫師成品：

```bash
python3 tools/generate_placeholder_sprites.py
```

輸出：角色（室內 + 等距）、地面 tile、建築、植被、農田的 placeholder PNG。
